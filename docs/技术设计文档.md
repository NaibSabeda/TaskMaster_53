# TaskMaster 技术设计文档  
## 一、项目架构  
### 1. 核心类设计  
- **TaskMaster类**：封装所有任务管理功能，包含方法：  
  - `load_tasks()`：加载本地JSON文件中的任务；  
  - `save_tasks()`：将任务保存到JSON文件；  
  - `add_task()`/`delete_task()`：任务增删；  
  - `list_tasks()`/`filter_tasks()`：任务查询与筛选；  
  - `complete_task()`：标记任务完成。  

### 2. 数据存储设计  
- 采用JSON文件（`tasks.json`）存储任务数据，结构示例：  
  ```json
  [
    {
      "id": 1,
      "title": "测试任务",
      "priority": "high",
      "deadline": "2025-12-01",
      "description": "测试内容",
      "completed": false,
      "created_at": "2025-11-25 10:00:00"
    }
  ]
